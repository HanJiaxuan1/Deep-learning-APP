Sud.DL Does it faster & better.

5 steps to deploy your own model easily.

The sud.DL deep learning deployment website supports 6 types of NLP deep learning models, including Text Classification, Translation, Text Generation, Summarization, Question Answering, and Fill Mask. When you want to deploy your deep learning model, please select the type of your model first, which will facilitate sud.DL to match the appropriate pipeline on the backend to process your model.

After selecting the type of model, you also need to fill in some text descriptions related to the model to ensure that users understand the functionality, limitations, potential deviations, and ethical considerations of your model.

Model Name: Give your model a descriptive name so that it can be recognized by you or users. Please note that the model name must be lowercase and have no spaces
Background: This section provides a detailed description of the source, purpose, and fine-tuning of the model based on which pre trained model.
Model Input: Provide a detailed description of the input formats and types required for the model, so that users can understand how to interact with the model.
Expected output: Clearly describe the output of the model, including output type, format, etc.

We offer two options for uploading models. The first option is to support you in uploading all files related to your model.
At the same time, you need to provide specific version information for training the model, including PyTorch version and Transformers version. This will help Sud.DL create your unique Docker environment for your single room, ensuring the safe operation of your model.

Usually, a complete model file will include the following five types of files:
Config.json
Pytorch model.bin
Tokenizer.json
Tokenizer_ Config.json
Vocab.txt

We offer two options for uploading models. The second option is to support you to only upload your adjusted model files, that is, upload pytorch separately_ Model. bin file.

At the same time, you need to provide the model of the parent model of this model, which is the model from which you fine-tuned your model. And it is still necessary to provide specific version information for training the model, including PyTorch version and Transformers version. This will help Sud.DL create your unique Docker environment for your single room, ensuring the safe operation of your model.

Now that everything has been uploaded, you can easily deploy your deep learning model by clicking the button! This may take 5-10 minutes. You can browse other content first, and we will prompt you after the deployment is completed. You will be able to see the model you uploaded on your personal page or using the model page.
